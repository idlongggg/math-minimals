# 🔧 Scripts

Chứa các script tự động hóa và migration.

## Files

- `auto-migrate.sh` - Script tự động migration
- `migrate-dashboard-layout.sh` - Migration layout dashboard
- `migrate-to-custom-components.sh` - Migration sang custom components

## Usage

Chạy script:
```bash
chmod +x scripts/<script-name>.sh
./scripts/<script-name>.sh
```
